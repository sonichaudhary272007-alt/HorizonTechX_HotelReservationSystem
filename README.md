# Hotel Management System

A console-based Hotel Management System developed in Java using Object-Oriented Programming (OOP) concepts and File Handling.

## Features
- User Login Authentication
- View Available Rooms
- Room Categories:
  - Standard
  - Deluxe
  - Suite
- Room Booking System
- Booking Cancellation
- Payment Simulation
- Date-wise Booking
- Booking Details Management
- Total Booking Counter
- File Handling using Java File I/O

## Technologies Used
- Java
- OOP Concepts
- File Handling
- ArrayList
- Eclipse IDE

## Project Structure
```text
HotelReservationSystem.java
```

## How to Run
1. Open Eclipse IDE
2. Create a Java Project
3. Create package:
```text
hotelreservation
```
4. Create file:
```text
HotelReservationSystem.java
```
5. Paste the code
6. Run the program
